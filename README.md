# Probability & Statistics for Data Science

This repository contains Python implementations of key probability
and statistical concepts used in data science and machine learning.

## Topics Covered
- Bayes Theorem
- Normal Distribution
- Binomial Distribution
- Hypothesis Testing
- Expected Value & Variance

## Features
- Clean Python implementations
- Mathematical understanding of concepts
- Visualizations for distributions
- Applications in machine learning

## Tools Used
- Python
- NumPy
- Matplotlib
- SciPy

## Structure
- notebooks/ → all implementations
- assignments/ → original questions (PDF)
